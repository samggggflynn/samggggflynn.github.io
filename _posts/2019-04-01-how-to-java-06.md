## 创建数组

数组是一个**固定长度**的，包含了**相同类型**数据的**容器**

### 声明数组

`int[] a;`声明了一个数组变量。
`[]`表示该变量是一个数组
`int`表示数组里的每一个元素都是一个整数
`a`是变量名
但是，仅仅是这一句**声明，不会创建数组**

有时候也会写成`int a[];`没有任何区别

```
public class HelloWorld{
	public static void main(String[] args){
		//声明一个数组
		int[] a;
	}
}
```

### 创建数组

创建数组的时候，要指明数组的长度；
`new int[5]`

**引用的概念:**
如果变量代表一个数组，比如a，我们把a叫做**引用**
与基本类型不同
`int c = 5;`这叫给c**赋值**为5

声明一个引用 `int[] a;`
`a = new int[5]；`
让a这个引用，**指向**数组

```java
public class HelloWorld{
	public static void main(String[] args){
		//声明一个引用
		int[] a;

		//创建一个长度是5的数组，并且使用引用a指向该数组
		a = new int[5];

		int[] b = new int[5];//声明的同时，指向一个数组
		
	}
}
```

### 访问数组

数组下标**基0**

下标0，代表数组里的第一个数

```java
public class HelloWorld{
	public static void main(String[] args){
		int[] a;

		a[0] = 1;//下标0，代表数组里的第一个数
		a[1]= 2;
        a[2]= 3;
        a[3]= 4;

```

### 数组长度

**.length属性**用于访问一个数组的长度
数组访问下标范围是0到长度-1
一旦超过这个范围，就会产生数组下标越界异常

```java
public class HelloWorld{
	public static void main(String[] args){
		int[] a;
		a = new int[5];//先声明一个引用，再把引用指向数组

		System.out.println(a.length);//打印数组的长度

		a[4] = 100;//下标4，实质上是“第五个”，即最后一个
		a[5] = 101;//下标5，实质上是”第六个“，超出范围，产生数组下标越界
		
	}
}
```

## Java数组选择法，冒泡排序

### 选择法排序

选择法排序的思路

**把第一位**和其他所有的进行比较，只要比第一位小的，就换到第一个位置来
比较完之后，**第一位就是最小的**
然后再从**第二位**和剩余的其他所有进行比较，只要比第二位小的，就换到第二个位置来
比较完之后，**第二位就是第二小的**

以此类推

![](/styles/images/2019-03-22-how2java/6.1.png)

```java
public class HelloWorld{
	public static void main(String[] args){
		int[] a = new int[]{18,62,68,82,65,9};
		//排序前先把内容打印出来
		for(int i = 0;i<a.length;i++){
			System.out.println(a[i]+" ");
		}
		System.out.println("\n");
		//选择排序法

		//第一步：先把第一位和其他所有位进行比较
		//如果发现其他所有位置的数据比第一位小，就进行交换

		for(int i = 1;i<a.length;i++){
			if(a[i]<a[0]){
				int temp = a[0];
				a[0] = a[i];
				a[i] = temp;
			}
		}
		//把内容打印出来
		//可以发现，最小的一个数，到了最前面
		for (int i = 0; i < a.length; i++) {
            System.out.print(a[i] + " ");
        }
        System.out.println("\n");

        //第二步：把第二位的和剩下的所有位进行比较
        for(int i = 2;i<a.length;i++){
			if(a[i]<a[1]){
				int temp = a[1];
				a[1] = a[i];
				a[i] = temp;
			}
        }
        //把内容打印出来
        //可以发现，倒数第二小的数，到了第二个位置
        for(int i = 0;i<a.length;i++){
			System.out.println(a[i]+" ");
        }
        System.out.println("\n");

        //可以发现一个规律
        //移动的位置是从0逐渐增加的
        //所以可以在外面套一层循环

        for(int j = 0;j<a.length-1;i++){
			for(int i = j+1;i<a.length;i++){
				if(a[i]<a[j]){
					int temp = a[j];
					a[j] = a[i];
					a[i] = temp;
				}
			}
        }

        //把内容打印出来
        for(int i = 0;i<a.length;i++){
			System.out.println(a[i]+" ");
        }
        System.out.println("\n");
	}
}
```

### 冒泡法排序

冒泡法排序的思路：
 第一步：从开始，把相邻的两位进行比较
 如果发现后面的比前面的大，就把大的数据交换在后面，循环比较完毕后，**最后一位就是最大的**
 第二步：再来一次，只不过不用比较最后一位
以此类推

![](/styles/images/2019-03-22-how2java/6.2.png)

```java
public class HelloWorld{
	public static void main(String[] args){
		int[] a = new int[]{18,62,68,65,9};
		//排序前，先把内容打印出来
		for(int i =0;i<a.length;i++){
			System.out.println(a[i]+" ");
		}
		System.out.println("\n");
		//冒泡法排序

		//第一步：从第一位开始，把相邻的两位进行比较
		//如果发现前面的比后面的大，就把大的数据交换在后面

		for(int i = 0;i<a.length;i++){
			if(a[i]>a[i+1]){
				int temp = a[i];
				a[i] = a[i+1];
				a[i+1] = temp;
			}
		}
		//把内容打印出来
		//可以发现，最大的到了最后面
		for(int i = 0;i<a.length;i++){
			System.out.println(a[i]+" ");
		}
		System.out.println("\n");

		//第二步：再来以此，只不过不用比较最后一位
		for(int i = 0;i<a.length-2;i++){
			if(a[i]>a[i+1]){
				int temp = a[i];
				a[i] = a[i+1];
				a[i+1] = temp;
			}
		}
		//把内容打印出来
		//可以发现，倒数第二大的到了导数第二个位置
		for(int i = 0;i<a.length;i++){
			System.out.println(a[i]+" ");
		}
		System.out.println("\n");

		//可以发现一个规律
		//后边界在收缩
		//所以可以在外面套一层循环

		for(int j = 0;j<a.length;j++){
			for(int i = 0;i<a.length-j-1;i++){
				if(a[i]>a[i+1]){
					int temp = a[i];
					a[i] = a[i+1];
					a[i+1] = temp;
				}
			}
		}

		//把内容打印出来
		for(int i = 0;i<a.length;i++){
			System.out.println(a[i]+" ");
		}
		System.out.println("\n");
	}
}
```

### 练习-排序

首先创建一个长度是5的数组,并填充随机数。 

首先用选择法正排序，然后再对其使用冒泡法倒排序

注 所谓的正排序就是从小到大排序，倒排序就是从大到小排序

答案：
```java
public class HelloWorld {
    public static void main(String[] args) {
        int[] a = new int[5];
        for (int i = 0; i < a.length; i++) {
            a[i] = (int) (Math.random() * 100);
            System.out.print(a[i] + " ");
        }
        System.out.println("\n");
        // 选择排序法
        for (int i = 0; i < a.length - 1; i++) {
            for (int j = i + 1; j < a.length; j++) {
                int temp = a[i];
                if (a[i] > a[j]) {
                    a[i] = a[j];
                    a[j] = temp;
                }
            }
        }
        System.out.println("正排序數組：");
        for (int i = 0; i < a.length; i++) {
            System.out.print(a[i] + " ");
        }
        // 冒泡排序法
        for (int i = 0; i < a.length - 1; i++) {
            for (int j = 0; j < a.length - 1 - i; j++) {
                int temp = a[j];
                if (a[j] < a[j + 1]) {
                    a[j] = a[j + 1];
                    a[j + 1] = temp;
                }
            }
        }
        System.out.println("\n倒排序數組：");
        for (int i = 0; i < a.length; i++) {
            System.out.print(a[i] + " ");
        }
    }
}
```
